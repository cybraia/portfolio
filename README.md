## 💻 Local Development

Get the portfolio running on your local machine:

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Run the Dev Server:**
   ```bash
   npm run dev
   ```
   Open `http://localhost:3000` to view the portfolio.

3. **Build for Production:**
   ```bash
   npm run build
   ```
